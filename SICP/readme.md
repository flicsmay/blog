###关于编译器
个人是选用drracket，感觉drracket会比mit scheme用起来方便多了。

头文件包是 `#lang planet neil/sicp`。
直接在drracket中输入头文件包然后重新打开就行了。
当代码多时使用drracket会比较卡。这时候就可以注释掉头文件再在其它文件中
`(load path)`就可以导入了，不过要写个安装函数（不然会报错），还是挺麻烦的，有更好的方法也欢迎告诉我。
